[+] Auction Mail Verification — Document Automation Workflow ✔ Complete AI-assisted document parsing for auction compliance

Designed a document automation workflow to streamline PLN/NOL Certificate of Mailing verification for storage unit auction files. The previous process required manually referencing mailing certifications against tenant records one at a time, which was slow and error prone.

Reduced manual lookup time by integrating AI-assisted document parsing with a documented process. The workflow standardizes how verification data is matched and extracted, cutting down what used to take hours into a matter of minutes. Built the process to be reproducible so that anyone on the team can follow it and maintain auction compliance.

[+] AWS Docs RAG — AI Documentation Search ⚙ In Progress NEW! UNDER CONSTRUCTION

If you've ever tried to find a specific answer in the AWS documentation, you know it can be rough. There are thousands of pages spread across dozens of services, and sometimes you just need a straight answer to a straightforward question. That's what this project is about.

I'm building an AI-powered search tool that uses Retrieval Augmented Generation (RAG) to pull answers directly from official AWS documentation. The idea is that instead of getting a generic AI response that might be wrong or outdated, you get an answer that's actually backed by the updated docs, with references you can check. I started this because I kept running into the same frustration while studying for the Cloud Practitioner exam.

The plan is to use Python for the backend, probably LangChain for the RAG pipeline, and a database like Chroma to store the document embeddings. The docs will be pulled from official AWS docs. This is still in the early stages, so nothing is deployed yet. I'll update this section as it comes together. It's a learning project as much as anything else, but it solves a real problem I've ran into.
